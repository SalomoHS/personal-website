# Personal Portfolio

This is my personal website.

## Tech Stack
- Next.js
- Tailwind CSS
- Supabase

## Features
- **Home**: Landing page with a hero section, summary statistics, and a call to action.
- **About**: Detailed profile including:
    - Personal introduction and bio
    - Career journey and education
    - Skills showcase
    - Achievements and hobbies
    - Photo gallery
- **Works**: Portfolio of projects with filtering (by category/tags) and sorting capabilities.
- **Contact**: A contact form and social media links for getting in touch.
