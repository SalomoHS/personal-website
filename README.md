The application is fully responsive and optimized for all devices, from mobile phones to large desktop screens.

- **Mobile-First Approach**: Ensures a seamless experience on smaller screens with touch-friendly interfaces.

### ü§ñ AI Chatbot ("Angkasa")
An intelligent virtual assistant integrated globally across the application to assist visitors.
- **Context-Aware**: Provides personalized greetings and assistance based on the page you are viewing.
- **Rich Interaction**: Supports Markdown rendering for formatted responses and features a "typing" effect for a natural feel.
- **Animated Avatar**: Includes a dynamic character that reacts to interaction states.
- **Floating Interface**: Easily accessible without obstructing the main content.
